# 摸鱼搭子桌宠

这是一个基于 `tkinter` 的桌面级桌宠，不需要额外安装第三方依赖。

## 当前能力

- 透明背景、无边框、始终置顶
- 左键点击互动，双击喂零食，右键打开自绘菜单
- 拖拽时有 Q 弹拉伸和局部鼓包效果，像掐脸一样会变形
- 支持多种桌宠类型：`团子`、`乌鲁鲁`、`皮卡丘`
- 会根据状态切换表情和动作：开心、眨眼、坏笑、吃东西、困倦、委屈、被捏变形等
- 点击后会弹出状态词条：心情、精力、专注

## 运行

双击：

- `start_deskpet.bat`

或者命令行运行：

```powershell
python office_pet.py
```

## 操作

- `左键单击`：互动
- `左键拖拽`：拖动桌宠
- `左键双击`：喂零食
- `右键`：打开菜单 / 切换宠物
- `Esc`：退出
