# 摸鱼搭子桌宠

这是一个用 Python 标准库 `tkinter` 写的桌面级摸鱼搭子，不需要额外安装第三方依赖。

## 功能

- 透明背景、无边框、始终置顶的小桌宠窗口
- 左键点击摸摸它，双击喂零食
- 右键菜单可以切换巡逻、贴边停靠、切到上班/摸鱼状态
- 它会自己在桌面上溜达，心情、精力、专注度会随时间变化
- 精力过低会自己睡觉，恢复后继续巡逻

## 运行

在当前目录双击下面这个文件即可：

- `start_deskpet.bat`

或者用命令行运行：

```powershell
python office_pet.py
```

## 操作

- `左键单击`：摸摸它
- `左键拖动`：移动桌宠
- `左键双击`：喂零食
- `右键`：打开桌宠菜单
- `Esc`：退出
