# Changelog

## 0.0.2

- Start each run from the beginning of the current visible editor range instead of line 1.
- Added ghost enemies that chase the player.
- Added lives and game-over handling.
- Reworked rendering into a serialized queue to avoid rapid-move crashes.

## 0.0.1

- Initial playable prototype.
- Toggle the game with `Ctrl+D`.
- Move with `Alt+Arrow` keys.
- Restore code instantly with `Ctrl+D` or exit with `Esc`.
- Added bite impact feedback with burst frames and combo status pulses.
